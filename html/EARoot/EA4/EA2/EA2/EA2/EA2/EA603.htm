Basic Path Basic Path
Details:
Steps:
1. A Secretária escolhe acessar seu perfil.
2. O Sistema apresenta a tela de perfil de Secretária.
3. A Secretária escolhe o ambiente de gerenciamento de matrícula
4. O Sistema apresenta o ambiente de gerenciamento de matrícula.
5. A Secretária escolhe matricular um aluno.
6. O Sistema apresenta as turmas existentes.
7. A Secretária escolhe uma turma.
8. O Sistema apresenta as informações necessárias à matrícula.
9. A Secretária responde as informações e escolhe criar a matrícula.
10. O Sistema confirma a matrícula e apresenta o ambiente de gerenciamento de matrícula.
Exceptions:
5a. A Secretária escolhe editar uma matrícula.
5b. A Secretária escolhe visualizar as matrículas.
