Basic Path Basic Path
Details:
Steps:
1. A Coordenadora escolhe acessar o seu perfil.
2. O Sistema apresenta o perfil de Coordenadora.
3. A Coordenadora escolhe o ambiente de gerenciamento de definição de aula.
4. O Sistema apresenta o ambiente de gerenciamento de definição de aula.
5. A Coordenadora escollhe uma data.
6. O Sistema apresenta as matérias, o local, e uma turma.
7. A Coordenadora responde as informações e escolhe definir a aula.
8. O Sistema confirma a definição da aula e apresenta o ambiente de definição de aula.
Exceptions:
5a. A Coordenadora escolhe editar aula.
5b. A Coordenadora escolhe excluir aula.
5c. A Coordenadora escolhe visualizar aula.
7a. A Coordenadora escolhe criar um local novo.
A Coordenadora escolhe criar um local novo. Alternate
Details:
Steps:
1. O Sistema pergunta qual é o novo local.
2. A Coordenadora responde e escolhe criar o novo local.
