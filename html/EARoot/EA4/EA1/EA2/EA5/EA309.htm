Basic Path Basic Path
Details:
Steps:
1. A Coordenadora escolhe acessar o seu perfil.
2. O Sistema apresenta o perfil de Coordenadora.
3. A Coordenadora escolhe o ambiente de gerenciamento de criação de turma.
4. O Sistema apresenta o ambiente de gerenciamento de criação de turma.
5. A Coordenadora escolhe criar turma.
6. O Sistema apresenta as informações necessárias para criação de turma.
7. A Coordenadora responde as informações e escolhe criar a turma.
8. O Sistema confirma e apresenta o ambiente de gerenciamento de criação de turma
Exceptions:
5a. A Coordenadora escolhe editar turma.
5b. A Coordenadora escolhe excluir turma.
5c. A Coordenadora escolhe visualizar turma.
