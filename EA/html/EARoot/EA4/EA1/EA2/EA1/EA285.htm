Basic Path Basic Path
Details:
Steps:
1. A Coordenadora escolhe acessar o seu perfil
2. O Sistema apresenta o perfil de Coordenadora.
3. A Coordenadora escolhe o ambiente de gerenciamento de curso.
4. O Sistema apresenta o ambiente de gerenciamento de curso.
5. A Coordenadora escolhe criar um curso.
6. O Sistema apresenta as informações necessárias para criação do curso.
7. A Coordenadora responde as informações e escolhe criar o curso.
8. O Sistema confirma a criação do curso e apresenta o ambiente de gerenciamento de curso.
Exceptions:
5a. A Coordenadora escolhe Visualizar o Curso.
5b. A Coordenadora escolhe Excluir Curso.
5c. A Coordenadora escolhe Editar Curso.
