Object Type Connection Notes
«Functional» R002 - O sistema DEVE atribuir um número univoco de três digitos de identificação sempre que um curso for cadastrado conforme RN005. Requirement Dependency  
«Functional» R001 - O sistema DEVE receber matriculas de alunos seguindo os criterios estabelecidos nas regras: RN - 002, RN - 003. Ao matricular um aluno o sistema DEVE criar um novo número de matrícula conforme a regra RN004. Requirement Dependency  
«Functional» R003 - O sistema DEVE atribuir um número univoco de identificação ao ser criar uma turma segundo a regra RN006. Requirement Dependency  
«Functional» R005 - O sistema DEVE avisar a coordenadora de ensino quando houver atividades pendentes de aprovação, conforme RN008, juntamente com as datas disponiveis segundo a regra RN009. Após a aprovação e definição de data o sistema deve informar o educador pro Requirement Dependency  
«Functional» R004 - O sistema DEVE verificar se não exista conflito de horários e salas ao definir uma aula . E deve vincular esta aula aos cursos a que for atribuída conforme RN007. Requirement Dependency
